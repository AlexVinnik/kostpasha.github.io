Вкладка «Разделы»¶

Вкладка разделы содержит кнопки на остальные таблицы требуемые для проекта, а именно:
  • Таблица «Сотрудники»
  • Таблица «Группы»
  • Таблица «Отделы»
  • Таблица «Должности»
  • Таблица «Руководители»

Как открыть таблицу¶

Для того что бы открыть нужную таблицу откройте в панеле вкладку «Разделы», после выберите нужную таблицу и нажмите на нее (будет открыто новое окно). В данном окне будет обтображаться информация по данному разделу. Внизу окна доступны три кнопки, работа которых будет описана далее. Работа кнопок будет приведена на примере таблицы «Сотрудники». Для остальных таблиц действие кнопок аналогичное.

Добавление записи¶

  1. Нажмите на кнопку добавить, что бы добавить нового сотрудника (будет открыто новое окно)
  2. В открывшемся окне заполните необходимую информацию для создания сотрудника
  3. После создания, нажмите кнопку «ОК» (окно будет закрыто и инфорамция отобразится в таблице)

Изменение записи¶

  1. Нажмите на того сотрудника информацию которого хотите изменить
  2. После нажмите кнопку «Изменить» (будет открыто новое окно)
  3. В окне измените данные тех параметров, которые вам нужны
  4. Нажмите кнопку «ОК» и ваши изменения будут применены

Удаление записи¶

  1. Нажмите на того сотрудника информацию о котором хотите удалить
  2. После нажмите кнопку «Удалить» (сотрудник будет удален)

Такая работа кнопок распространяется на все оставшиеся таблицы. Для того что бы вернуться на главное окно, закройте текущее окно.
